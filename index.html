A propos de ce site

Ce site a pour objectif de présenter quelques aspects de mon travail, de mes compétences et servir de laboratoire d'expérimentation pour ce qui concerne la programmation web et particulièrement les langages de balisages (HTML5, CSS3) pour formater et mettre en forme le contenu, du JavaScript côté client pour les aspects interactifs, des APIs pour garantir de la persistance dans le contenu, de la synchronisation en temps réel et ainsi de suite...

Selon la traduction faite de la JAMStack (JavaScript, APIs and Markup, voir ici : jamstack.org)Les navigateurs sont le nouveau système d’exploitation. Les serveurs sont absorbés par les CDns et les APIs. Les sites sont ou dépourvus de base de données et générés en amont ou font appel à de services de bases de données hébergées et servies directement depuis le navigateur. JavaScript est en charge de toute programmation dynamique pendant le cycle requête/réponse et tourne entièrement côte client. Ce site est dit "statique" et est volontairement hébergé sur GitHub car le code source de ce site est versionné dans git. La partie "dynamique" est géré par le Javascript qui s'éxecute donc côté client.

N'étant pas développeur, je tire profit des frameworks open-source existant, et ce site est construit à partir de Bootstrap et HTML5 Boilerplate. Les librairies JS sont hébérger sur un CDN pour la performance

Enfin les librairies utilisées seront celles de Leaflet et d'OpenLayers. A l'heure actuelle, seule Leaflet est intégrée (ci-dessous et dans la rubrique Blog)

Un aperçu de mon environnement professionnel

QGIS

QGIS est un Système d'Information Géographique (SIG) distribué sous licence publique générale GNU. C'est un projet officiel de la fondation Open Source Geospatial (OSGeo). Il est compatible avec Linux, Unix, Mac OS X, Windows et Android et intègre de nombreux formats vecteur, raster, base de données et fonctionnalités.

PostgreSQL

PostgreSQL est un système de gestion de base de données relationnelle et objet (SGBDRO) libre sous licence BSD et comporte de nombreuses extensions dont une gestion des données géographiques via PostGIS

FME

ETL (Extract Transform Load) qui gère une multitude de formats en entrée comme en sortie dont la totalité des formats spatiaux. Un de mes principaux outils pour le traitement de données géographiques. La version française est distribué par VEREMES qui assure la traduction, le support et quelques extensions open-source.

Adobe CS5

La suite logicielle d'Adobe pour le graphisme : Photoshop et Illustrator pour la reprise et les effets cartographiques et InDesign pour des mises en page poussée et la publication de maquette et support de communication
